Why a colophon? A bit of a homage, really. I was a yearbook nerd in high school and worked for the CU yearbook my junior and senior years of college (one of which I was the Editor-in-Chief). On top of that, I spent a couple years working for the CU student newspaper as a reporter and, later, as the online editor (I even met my wife working for the paper!).

Typography

All fonts are provided by Google Fonts. All headers, the main navigation and site title use Open Sans. The main body copy uses Merriweather.

Hardware

All web development is done on my 15" Macbook Pro because a *nix system is a necessity for productive web development.

Software

All final HTML files are compiled by Jekyll, a Ruby-based static site generator. Production CSS is produced with Sass, one of the best things to happen to front end web development in recent years. Local Ruby versions are managed by RVM because counting on your system version of Ruby is crazy. Local Ruby Gems and dependencies are managed by Bundler - a great way to take the guesswork out of managing Ruby Gems and Gemsets.

Publisher

The git repository for this site is available publicly through Github. Compiled code is hosted with Arvixe, arguably the best cheap shared hosting provider I've found.

Content

All site content is original and produced by me, Nick Switzer, unless otherwise mentioned.
